# Web API

Starter template pro Code to God Mode — ASP.NET Core Web API s controllery.

## Jak začít

```bash
dotnet restore
dotnet build
dotnet run --project src/Api
```

API poběží na `http://localhost:5000`. Swagger UI na `/swagger`.

## Testy

```bash
dotnet test
```

## Struktura

```
src/Api/             — API projekt (controllery, modely, služby)
tests/Api.Tests/     — testy
```
